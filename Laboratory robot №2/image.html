GitHub

GitHub, Inc. — це Інтернет-сервіс для розробки програмного забезпечення та контролю версій за допомогою Git . Він забезпечує розподілений контроль версій Git плюс контроль доступу , відстеження помилок, запити програмних функцій, керування завданнями, безперервну інтеграцію та вікі для кожного проекту. Штаб-квартира в Каліфорнії, з 2018 року є дочірньою компанією Microsoft.

Він зазвичай використовується для розміщення проектів розробки програмного забезпечення з відкритим кодом. Станом на січень 2023 року GitHub повідомляв про понад 100 мільйонів розробників і понад 372 мільйони сховищ, включаючи щонайменше 28 мільйонів публічних сховищ. Станом на листопад 2021 року це найбільший хост вихідного коду .
